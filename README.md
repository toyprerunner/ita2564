
# Yii2-ITA
================================
### Install
- สร้าง db ชื่อ ita
- นำเข้าข้อมูลไฟล์ `ita.sql`
- แตก zip file แล้ว cd เข้าไป /ita
จากนั้นทำการ composer update
```
composer update
```
- เปลี่ยน permission โฟลเดอร์ runtime, web/assets, web/resume, /web/uploads เปลี่ยนเป็น 777 ทั้งหมด
"# ITA"
